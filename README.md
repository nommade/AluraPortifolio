# Alura Portfólio

Este é um projeto de portfólio pessoal desenvolvido para apresentar habilidades, projetos e informações profissionais. O objetivo é fornecer uma página simples e elegante para destacar experiências e trabalhos realizados.

## Estrutura do Projeto

- `index.html`: Página principal do portfólio.
- `about.html`: Página sobre mim.
- `assets/`: Pasta para imagens, ícones e outros recursos estáticos.
- `styles/style.css`: Arquivo de estilos CSS para o site.

## Como Visualizar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador preferido.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Personalização

Você pode editar os arquivos HTML e CSS para adicionar suas próprias informações, projetos e estilos.
